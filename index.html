Sharing Reports as Websites

The default output format for new R Markdown scripts created with RStudio is HTML (with the content saved in a .html file). HTML stands for HyperText Markup Language. This means that the .html files you create with R Markdown can be put online as webpages for others to view!

As it turns out, you can use GitHub not only to host versions of your code repository, but also to serve (display) .html filesâincluding ones generated from R Markdown. This feature is known as GitHub Pages.

  • Next, you need to configure that GitHub repository to enable GitHub Pages.
